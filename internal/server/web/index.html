oh-my-lan

连接服务器

请填写 omlserver 控制平面地址，例如 http://vps.example.com:8080。

登录

使用 omlserver admin user set 创建的账号密码登录。

设备离线推送（bark）

server 周期检测设备 last_seen_at，超过阈值时把告警推到你的 bark URL （iOS/macOS 端 App：Bark）。 每次离线只推一次，回到在线后重置。

服务端日志

server 进程的最近运行时日志（环形 buffer，容量 1000 条）。每次展开会重新拉一次。 重启 server 缓冲清空。
